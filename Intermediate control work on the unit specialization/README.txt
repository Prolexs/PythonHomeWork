Программа представляет собой консольное приложение для работы с заметками. Она обладает следующим функционалом:

    Создание заметки: Пользователь может создать новую заметку, указав заголовок и текст заметки. Опционально можно добавить теги к заметке.

    Просмотр всех заметок: Пользователь может просмотреть список всех созданных заметок. Выводятся заголовки, тексты и даты создания заметок.

    Редактирование заметки: Пользователь может выбрать заметку для редактирования, указав ее ID. Затем пользователь может изменить заголовок и текст заметки.

    Удаление заметки: Пользователь может удалить заметку, указав ее ID.

    Поиск заметок: Пользователь может выполнить поиск заметок по ключевому слову. Выводятся заметки, в заголовках или текстах которых присутствует указанное ключевое слово.

    Сортировка заметок: Пользователь может отсортировать заметки по дате создания (по умолчанию) или по дате изменения. После сортировки заметки выводятся в соответствующем порядке.

    Добавление тегов к заметке: Пользователь может добавить теги к выбранной заметке, указав ее ID.

    Экспорт заметок в CSV: Пользователь может экспортировать все заметки в файл CSV. Экспортируются заголовки, тексты, даты создания и теги заметок.

Кроме того, программа обеспечивает стабильную работу и обработку возможных исключений. Например, она проверяет наличие файла с заметками при загрузке, обрабатывает ошибки при сохранении заметок в файл и экспорте в CSV, а также предусмотрена обработка ошибок ввода данных пользователем.

Все заметки сохраняются в формате JSON в файл с именем "notes.json". Это позволяет сохранять данные между запусками программы и обеспечивает их сохранность.

Программа предоставляет удобный способ управления заметками через консольный интерфейс, позволяя пользователям создавать, просматривать, редактировать, удалять, искать, сортировать и экспортировать заметки.
